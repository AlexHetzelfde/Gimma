Welkom bij WikiLeer

Om lessen te genereren heb je een gratis Gemini API key nodig.
Haal hem op via Google AI Studio — geen creditcard nodig. De key wordt alleen lokaal opgeslagen in jouw browser, nooit verstuurd naar een server.

API key wijzigen

Je huidige key wordt vervangen. De les-cache van vandaag blijft bewaard.

⚠ Als je opnieuw begint, worden de opgeslagen antwoorden van deze les gewist. Eerdere lessen blijven bewaard.
🏠
Les afsluiten? Je voortgang wordt bewaard — je kan op elk moment verder waar je gebleven bent.
